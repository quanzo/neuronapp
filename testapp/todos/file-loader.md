1. Определи путь к файлу `temp/671.md`
2. Определи количество строк в файле
3. Начни читать и обрабатывать файл по частям. Начни с start_line = 0. Используй размер чанка в 100 строк. Максимальный размер чанка в символах = 10000
4. Читай блок файла с помощью `view_chunk`
5. Для каждого блока составь краткое резюме этого фрагмента (summary). Старайся выделить ключевые идеи, факты, темы.
6. Сохрани полученное резюме под уникальным ключом, например, "part_0", "part_1" и т.д., используя `intermediate_pad`.
7. Увеличь start_line на размер прочитанного блока и продолжай, пока не будет достигнут конец файла (когда прочитано меньше запрошенного размера или блок пуст).
8. Определи достигнут ли конец файла. Если конец файла достугнут, то `перейди к пункту 9`. Если мы не достигли конца файла, то `перейди к пункту 4`.
9. Используй `intermediate_list` чтобы получить список сохраненных резюме.
10. Прочитай резюме, используя `intermediate_load`.
11. Перейди к следующему. Если список обработан не полностью, то `перейди к пункту 10`. Если достиг окончания списка, то `перейди к пункту 12`.
12. По текстам из резюме собери общий вывод.
13. Выведи результат.
