Твоя задача, используя `chunk_grep` найти в файле все блоки/чанки с интересующим контентом.

Входные данные:
- Запрос на поиск: `query` = "$query"
- Файл в котором ищем: `path` = "$path"


На основании `query` сформируй массив строк для релевантного поиска чанков в файле. `query` может содержать слова, которые не несут смысла и поэтому из поиска их убери. При составлении разных вариантов фраз для поиска можно применять синонимы.

Используй `chunk_grep` и передай массив строк для поиска чанков и путь к файлу.

Вызови инструмент `chunk_grep` и найди блоки с релевантным запросу контентом.
