/*
    Ищет в файле определенные фразы и выбирает блоки с этими фразами
*/

Твоя задача, используя `chunk_grep` найти в файле все блоки/чанки с интересующим контентом.

Входные данные:
- Запрос на поиск: `query` = "$query"
- Файл в котором ищем: `path` = "$path"

На основании `query` сформируй массив строк для релевантного поиска чанков в файле. `query` может содержать слова, которые не несут смысла и поэтому из поиска их убери. При составлении разных вариантов фраз для поиска можно применять синонимы. Вариантов должно быть не менее 3.

Составь поисковую строку, в которой релавантные варианты поиска будут разделены символом `|`. Поисковая строка должна быть одна!

Вызови инструмент `chunk_grep` и найди блоки с релевантным запросу контентом.

Обработай поступившие данные. Выводы делать не надо. Выведи детальную информацию по теме запроса.
