# Skill - навыки и умения

Язык программирования: php 8.1

# Структура каталогов

Все пути к каталогам заданы от корня проекта

* `classes/` - здесь будем хранить классы
* `interfaces/` - здесь будут интерфейсы

Базовый namespace для классов `app\modules\neuron\classes`

Базовый namespace для интерфейсов `app\modules\neuron\interfaces`

Все классы для реализации текущего проекта разместить в `classes/skill`

# Структура

Каждый навык представляется классом `Skill` и реализует интерфейс `ISkill`

## класс Skill

Конструктор класса принимает текст.

Формат текста такой же как и для `TodoList`, только в `Skill` одно задание. И также поддерживаются options.

В тексте skill допускается использовать символы подстановки вида `$1` или `$2` что соответсвует номеру параметра, который передается при получении текста. Параметры передаем в метод `getSkill`.
