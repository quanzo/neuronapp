В классе `Skill` есть метод `getSkill` который надо переписать так, чтобы он принмал именованные параметры, а не нумерованные - у каждого параметра должно быть имя теперь. Учти, надо избежать коллизий  с именами при замене! И для имени параметра будет ограничение: латинские буквы без пробелов, учитываем регистр!

У NeuronAi есть `vendor/neuron-core/neuron-ai/src/Tools/Tool.php` - инструменты для LLM. Должна быть возможность генерировать инструмент LLM на основе `Skill`. Для этого надо сделать так, чтобы Skill стали поддерживать параметры.

Иными словами, у `Skill` должен появиться метод `getTool(): Tool` Где Tool это вот `vendor/neuron-core/neuron-ai/src/Tools/Tool.php` Имя инструмента: это имя файла с поддиректорией (если она есть). Описание инструмента: пусть это будет значение свойства из массива `options`. Также в `options` должны задаваться возможные параметры инструмента.


Метод `getTool` класса `Skill` - вызов иструмента не должен возвращать текст! а вызов инструмента должен вернуть результат работы LLM по этому тексту! И поэтому метод `getTool` должен как и метод `executeFromAgent` принять конфигурацию агента LLM и роль и на основании этого создать инструмент, который вернет результат выполнения, а не текст.


`TodoList` в настройках в свойстве `options` может содержать опцию `skills`. В файле с содержимым, на основании которого мы создаем TodoList, это так и будет называться skills - строка, в которой через запятую (возможно пробелы), будут пречислены Skill по именам, которые должны быть доступны при исполнении агентом LLM этого списка.

Создадим в TodoList метод `getNeedSkills(): array` который вернет имена навыков (skill), которые необходимо подключать при исполнении.

При вызове `executeFromAgent` класс `TodoList` агент получить инструменты. Для этого класс `Skill` реализует метод `getTool`. И обязательно: подключение инструментов в агент не должно изменять основное состояние - то есть после исполнения todolist состав инструментов должен быть прежним. Для этого, при подключении skill в качестве инструментов (tool) надо клонировать agentCfg; занулить свойство _agent; вызвать getAgent() чтобы класс пересоздался; затем добавить инструменты к агенту, который получили; вызвать sendMessage.
